# 🧮 Unique Calculator

A simple **modern and responsive calculator** built using **HTML, CSS, and Vanilla JavaScript**.  

This calculator supports **all basic arithmetic operations**, has **AC (All Clear)** and **DEL (Delete last character)** buttons, and a **modern glassmorphism UI**.  

It is fully **responsive**, works on mobile and desktop, and is completely **unique** with no copyright issues.

## ✨ Features

- Basic arithmetic: `+ − × ÷`  
- Decimal support `.`  
- All Clear (AC) button  
- Delete (DEL) button  
- Equals `=` button  
- Modern, clean UI  
- Fully responsive for mobile & desktop
